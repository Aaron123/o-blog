Adding icons

  •  Using modal source code
  • Using Bootstrap grid 

Icons from glyphicons are supported support by simply naming the icon using italic style:

Source example 

/icon-calendar/ calendar

Rendered output

icon-calendar calendar

This could also be used for the top navbar using something like:

Source example 

- /icon-book icon-white/ Documentation
  - Templates
  - Structures
  - Functions
  -
  - FAQ

- /icon-file icon-white/ Blog
  - 测试个新条目哦
  - Using Bootstrap grid
  - Adding icons
  - Using modal source code
  - Alerts

- /icon-tags icon-white/ Tags

Rendered output

  • icon-book icon-white Documentation
    • Templates
    • Structures
    • Functions
    • FAQ
  • icon-file icon-white Blog
    • 测试个新条目哦
    • Using Bootstrap grid
    • Adding icons
    • Using modal source code
    • Alerts
  • icon-tags icon-white Tags

Please note the blank line between menu items. if blanks are omitted, the rendered result might be damaged.
